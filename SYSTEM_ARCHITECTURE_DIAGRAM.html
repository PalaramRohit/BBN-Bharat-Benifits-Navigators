Core Modules & Responsibilities

🔄 Orchestrator

Coordinates all service interactions, manages workflow execution, and ensures proper sequencing of operations across decision and intelligence modules.

📋 Decision Engine

Core business logic for eligibility assessment. Works with agents to evaluate citizen data against scheme rules and generate recommendations.

🤖 Eligibility Agent

Analyzes citizen eligibility criteria using AI. Integrates with LLM service for intelligent assessment and RAG retriever for policy data.

📝 Explanation Engine

Generates human-readable explanations for decisions. Works with document advisor and explainability engine to provide transparent reasoning.

🧠 LLM Service

Manages interactions with Large Language Models. Used by agents for intelligent assessment and reasoning capabilities.

🛠️ RAG Retriever

Retrieves relevant context from knowledge base using semantic search. Provides policy information to decision and explanation modules.

🗺️ Pathway Detector

Identifies optimal benefit pathways. Returns recommended schemes and sequence of actions for citizen benefit.

⚡ Next Action Engine

Determines immediate action items for citizens. Guides them through workflow with clear next steps and requirements.
