2. File-by-File Breakdown

1
The Gateway
backend/app/main.py

This is the front door. It starts the Uvicorn server and tells FastAPI to listen for requests. It also connects to MongoDB as soon as the app starts.

2
The API Routes
backend/app/api/v1/endpoints.py

Think of this as the traffic police. When a user sends a query to /api/v1/query, this file receives it and hands it over to the Orchestrator.

3
The Brain (Orchestrator)
backend/app/orchestration/orchestrator.py

The most important file. It manages the whole conversation. It calls the search engine, then the rules engine, then the AI explainer in the right order.

4
The Search Engine
backend/app/utils/rag_retriever.py

This looks through all your 90+ JSON scheme files. It uses AI (Vector Search) to find the schemes that actually match what the user is talking about.

5
The Rule Matcher
backend/app/agents/eligibility_agent.py

This doesn't use AI—it uses logic. It checks: "Is the user's age > 60?" or "Is their income < 2 Lakhs?". It gives a simple Yes/No for every scheme found.

6
The Smart Explainer
backend/app/agents/explanation_agent.py

This is the Gemini AI. It takes raw data and turns it into a friendly message like "Namaste! Since you are a farmer in Telangana, you are eligible for..."

7
The Helper (Decision Layer)
backend/app/decision/decision_engine.py

This handles the extra info. It calculates the 85% success probability and checks which documents (Aadhaar, etc.) the user is still missing.

🛠️ Simple Execution Step

To run everything we've built, follow these 3 steps in your terminal:

  1. Activate: .\venv\Scripts\Activate.ps1
  2. Ingest Data: python -m backend.app.utils.bulk_processor (Already done!)
  3. Launch: python -m backend.app.main

Success! Your "Brain" is now alive at localhost:8000.
